Projects

De afgelopen maanden heb ik met veel enthousiasme gewerkt aan diverse opdrachten en projecten, zowel individueel als in teamverband. Elk project bood de kans om nieuwe tools en technieken te ontdekken en mijn vaardigheden in front-end ontwikkeling, design en het werken met API’s verder te ontwikkelen. Deze pagina geeft een overzicht van mijn werk en groei als ontwikkelaar. Van het namaken van bestaande websites tot het bouwen van interactieve toepassingen – elk project vertelt zijn eigen verhaal.

Best of 2024

2025

Best of 2024 is een nagebouwde website die ik heb ontwikkeld als oefenopdracht. Naast het namaken van de layout en styling, heb ik ook gebruikgemaakt van een API om dynamisch content op te halen via JavaScript

Front-end Portfolio

2025

Dit front-end portfolio is een moderne, responsive website die ik zelf heb opgebouwd met HTML en CSS. Het bevat een theme switcher (licht/donker modus) en is geoptimaliseerd met Webpack voor efficiënte bundling en performance. Alles is geschreven in schone, schaalbare CSS-structuur met aandacht voor design én toegankelijkheid.

Gamescon

2025

De Gamescom-website is een interactieve webpagina die ik volledig gebouwd heb in pure JavaScript voor het vak PGM-1. De site bevat onder andere een aftellende klok naar het evenement, en is opgezet zonder frameworks of externe libraries – puur JS, HTML en CSS.

STAM

2025

Deze website voor het STAM-museum is opgebouwd met HTML en CSS en focust op een duidelijke structuur en visueel aantrekkelijke layout. Het project toont hoe je met eenvoudige technieken een informatieve en gebruiksvriendelijke website kan opbouwen.

Funky Portfolio

2025

Funky Portfolio is een creatieve website die ik heb ontwikkeld voor het vak WEB-2. De site is volledig opgebouwd met HTML en CSS, met een sterke focus op animatie en visuele dynamiek. Alles draait om speelsheid, beweging en een opvallende gebruikerservaring.
